## ğŸ“¡ API Endpoints

### Get all students

```http
GET /students/
```

### Get student by ID

```http
GET /students/{id}
```

### Add student

```http
POST /students/
Content-Type: application/json

{
  "name": "Budi",
  "balance_saldo": 10000
}
```

### Update student

```http
PUT /students/{id}
Content-Type: application/json

{
  "name": "Andi",
  "balance_saldo": 20000
}
```

### Delete student

```http
DELETE /students/{id}
```

### Example with curl

```bash
# Get all students
curl -X GET http://127.0.0.1:8000/students/

# Add a new student
curl -X POST http://127.0.0.1:8000/students/ \
  -H "Content-Type: application/json" \
  -d '{"name": "Dodo Sidodo", "balance_saldo": 10000}'
  
#Update
curl -X PUT http://127.0.0.1:8000/students/1 \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Siti",
    "balance_saldo": 25000
  }'
```

### ğŸ‘‰ Keterangan:

* `PUT` â†’ method untuk update
* `http://127.0.0.1:8000/students/1` â†’ `1` adalah **id\_siswa** yang mau diubah
* `-H "Content-Type: application/json"` â†’ header supaya server tahu kita kirim JSON
* `-d` â†’ data body JSON

Kalau misalnya mau update **hanya salah satu field** (misal cuma `balance_saldo`), bisa gini:

```bash
curl -X PUT http://127.0.0.1:8000/students/1 \
  -H "Content-Type: application/json" \
  -d '{
    "balance_saldo": 50000
  }'
```
